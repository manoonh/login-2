# Simple Login Page 🔐

A clean and beginner-friendly **Login Page** built using **HTML & CSS**.
This project is simple, responsive, and perfect for beginners.

---

## 🚀 Features

* Clean and modern UI
* Responsive design
* Easy to customize
* Beginner-friendly code

---

## 🛠️ Tech Stack

* HTML5
* CSS3

---

## 📁 Project Structure

```bash
login/
│── index.html
│── style.css
└── README.md
```

---

## ▶️ Getting Started

Clone the repository:

```bash
git clone https://github.com/manoonh/login.git
```

Open `index.html` in your browser.
